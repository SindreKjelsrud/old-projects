## Details

***Run Minecraft server in "screen"***
```
1. Start screen with `screen`
2. Run start command: `java -Xmx4096M -jar server.jar nogui`
3. Press "ctrl + a + d" to exit screen
4. To enter the running screen, write `screen -r`
```
